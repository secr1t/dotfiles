# About

Personal dotfiles.

## Include

- kitty - terminal
- nvim - Neovim
- hyprl - Hyprland + Hyprlock
- yazi - file manager
- qutebrowser - keymap browser
- fastfetch - neofetch like 
- waybar - status bar
- fish - shell
